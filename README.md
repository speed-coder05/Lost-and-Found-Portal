# Lost-and-Found-Portal
This is a lost and found portal developed for institutions to digitally manage the process of retrieving and reporting lost and found articles on campus
This project developed a fully functional web platform to manage lost and found items. Users can register, submit items, upload images, and view all active listings. The platform streamlines the recovery process while providing a secure and easy-to-use interface.
All technical objectives, such as database integration, responsive design, user authentication, and image handling, were successfully implemented.
